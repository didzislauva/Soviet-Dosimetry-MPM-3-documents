# Soviet-Dosimetry-MPM-3-documents

# измеритель мощности дозы мрм-3

The MPM-3 (МПМ-3 / МРМ-3) is a 1974 Soviet dose rate meter for measuring X-ray and gamma radiation exposure using an ionization chamber. This scanned manual details technical specs, schematics, and maintenance for professional dosimetry in medical/industrial settings.
